# Code of Conduct

This project follows the **Contributor Covenant Code of Conduct**, version 2.1.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct.html

## Summary

We are committed to making participation in this project a welcoming experience for everyone. We expect all participants to act with empathy, kindness, and respect.

Unacceptable behavior may be reported to the project maintainers. All complaints will be reviewed and investigated promptly and fairly.
